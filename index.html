Todolist

Todolist - is a web service and set of task management software.

These tasks may contain notes, sticky notes. Tasks can be sorted by filters, assigned tags, and edited.

Stack: React, Redux, Typescript, Javascript, Formik, MUI, Axios, React-Router-dom

eShop

eShop - a website that sells goods via the Internet. Allows users online, in their browser or through a mobile application, to create a purchase order, select a method of payment and delivery of the order, and pay for the order.

The backend of the application is implemented using Firebase. It is possible to add the selected product to the cart, add and reduce the quantity of the product, or completely remove it from the cart. And also go to the payment page.

Stack: React, Redux, Typescript, Javascript, Firebase, MUI, Formik, React-Router-dom, Vite, Jest

Qwiz Cards

Qwiz Cards - is a time-tested technique for memorizing new foreign words, definitions, formulas, theorems and other necessary information using a flashcard test.

The application is implemented using RTK Qwery. To test the application, enter login: test@test.com and password: test.

Stack: React, RTK Qwery, Typescript, Javascript, Swagger, UI Kit, React Hook Form, React-Router-dom, Vite, Storybook
