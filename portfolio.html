Todolist

Todolist - is a web service and set of task management software.

These tasks may contain notes, sticky notes. Tasks can be sorted by filters, assigned tags, and edited.

Stack: React, Redux, Typescript, Javascript, Formik, MUI, Axios, React-Router-dom

Social Network

Social Network - a platform, online service or website designed to build, reflect and organize social relationships.

The application implements the ability to log in using the "React Hook Form". "CRUD" operations are performed using the axios library.

Stack: React, Redux, Typescript, Javascript, Axios, React Hook Form, React-Router-dom
